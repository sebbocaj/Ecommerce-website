<h1>React Ecommerce Website.</h1>

<p>You can Create, Read, Update and Delete products in the admin section.</p>

<p>You can dynamicly store image of your product to Cloudinary cloud management.</p>

<p>You can Register and Login as a customer.</p>

<p>You can Add product to your cart, Checkout and pay throught Stripe online payment, emails are send to the customer and the admin after each command.</p>

<h1>Stack</h1>
<p>I used Meteor, React.js, MongoDb, Cloudinary, Stripe</p>
